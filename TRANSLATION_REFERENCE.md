## Translation Strategy

### Hospitality Industry Terminology

The following terms were translated using industry-standard conventions:

| Concept | EN | FR | DE | NL | ES |
|---------|----|----|----|----|-----|
| Check-in | Check-in | Arrivée | Ankunft | Aankomst | Entrada |
| Check-out | Check-out | Départ | Abreise | Vertrek | Salida |
| Booking | Direct booking | Réservation directe | Direktbuchung | Directe boeking | Reserva directa |
| Deposit | Deposit (20%) | Acompte (20%) | Anzahlung (20%) | Aanbetaling (20%) | Depósito (20%) |
| Free cancellation | Free cancellation up to 7 days before | Annulation gratuite jusqu'à 7 jours avant | Kostenlose Stornierung bis 7 Tage vorher | Gratis annulering tot 7 dagen van tevoren | Cancelación gratuita hasta 7 días antes |
| Self check-in | Self check-in | Enregistrement autonome | Selbstcheck-in | Zelfcheck-in | Registro autónomo |
| Rooms | Rooms | Chambres | Zimmer | Kamers | Habitaciones |

### Meta Tag Strategy

**Title Tags:**
- Maintained consistent structure across all languages
- Included discount percentage for SEO impact
- Localized property name handling (Manfredonia appears in all)

**Meta Descriptions:**
- 155-160 characters (optimal for search results)
- Led with call-to-action (Book/Reserve/Buchen/Boek)
- Highlighted key benefits: savings %, self check-in, proximity to sea
- Included specific location (Gargano/Gargano region variations)

**Keywords:**
- Localized primary keyword variations
- Maintained intent (vacation rental, booking, rooms)
- Included geographic terms in each language
- Avoided over-optimization while preserving relevance

- All HTML structure remains identical to Italian version

### Dual Translation System
The page uses a hybrid approach:

1. **HTML Meta Tags & Static Content**: Translated in this HTML (visible to search engines and social media)
2. **Dynamic Content**: Handled by JavaScript i18n system (translates content on user interaction)

This ensures:
- ✓ Proper SEO with translated metadata
- ✓ Social media preview in correct language
- ✓ Dynamic UI translation via JavaScript
- ✓ Seamless language switching for users
