## Structured Data (Schema.org) Updates

### Updated Type Classifications
- **Changed from:** `"@type": "BedAndBreakfast"` (for room pages)
- **Changed to:** `"@type": "LodgingBusiness"` (more accurate for holiday rental)

### Why This Change?
- `BedAndBreakfast` specifically implies lodging with breakfast service
- `LodgingBusiness` is the correct schema type for holiday rentals/self-catering accommodations
- This improves SEO accuracy and helps Google properly categorize the property

### Descriptions Updated
All schema descriptions that mentioned "Camera B&B" have been updated to "Camera in casa vacanze" to maintain consistency.
